### 表情の意味

| ファイル名 | 表情 | 使われる場面 |
|---|---|---|
| `normal` | 平常 | 会話の序盤。まだ余裕がある |
| `annoyed` | 不機嫌 | 半目・眉間にしわ。渋っている |
| `angry` | 怒り | 目を見開き語気が強い。青筋 |
| `smug` | ドヤ顔・煽り | ニヤリと勝ち誇る。一番ムカつく顔に |
| `furious` | 激怒 | 顔面爆発寸前。湯気・怒髪天 |
| `defeated` | 敗北 | 「黙れカス」を食らって吹っ飛ぶ。白目/渦巻き目、のけぞり |

## 技術仕様

- **サイズ**: 1024×1024px(正方形)
- **形式**: PNG、**背景透過**
- **構図**: バストアップ(胸から上)、正面〜やや斜め。キャンバス内で顔が中央上寄り、下端は体で接地
- **差分の作り方**: 同一キャラの6表情は**輪郭・構図・体位置を完全に固定**し、表情(眉・目・口)と小物エフェクト(青筋・湯気・汗など)だけを差し替えてください。ゲーム内でクロスフェード切替するため、位置がずれるとちらつきます(defeatedのみ、のけぞり・傾きなどポーズが変わってOK)
- **画風**: 3キャラ共通のタッチで統一。デフォルメの効いたコミカルなアニメ調(ゲームの世界観がギャグ寄りのため、リアル調・劇画調は避ける)。線は太めではっきり、色数は絞ってフラット気味に
- **命名・納品**: `<キャラクターID>/<表情>.png`(例: `kessaisha/normal.png`)。この名前で `public/characters/` に置くだけでゲームに反映されます
